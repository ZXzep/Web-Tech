Web developer

A web developer is a programmer who specializes in, or is spe engaged in, the development of World Wide Web applications using a client-server model. The applications typically use HTML, CSS and Script in the client, PHP, ASP.NET (C#),Python or Jav and http for communications between client and server. content management system is often used to develop and maintain web applications.
